# 偉人マネジメント相性診断 MVP

上司・部下の関係性を円滑にするための、歴史上の人物モチーフ診断アプリです。採用判定ではなく、会話のきっかけと相互理解を目的にしています。

## Setup

```bash
pnpm install
pnpm dev
```

## MVP scope

- ひとり診断: 上司視点または部下視点で診断
- ペア診断: 上司と部下がそれぞれ回答し、摩擦ポイントと会話の提案を表示
- 6軸 × 5段階の定型フィードバック
- 日本と世界の歴史上の人物モチーフ
- キャラクター画像は未実装。後から `HistoricalCharacter.imagePrompt` または `imageSrc` を追加してください。

## Non-goals

- 採用、評価、昇進、処遇決定への利用
- 臨床診断、メンタルヘルス診断
- 実在人物の厳密な心理分析
